# docs/ - 五文件工作流

> 复杂项目的持久化执行文档

## 定位

- **作用**：存储项目规划需求，五文件工作流文档
- **更新**：项目执行时更新
- **生命周期**：跟随项目周期

## 五文件

| 文件 | 作用 |
|------|------|
| `prompt.md` | 项目定义（目标/排除项/交付物） |
| `plans.md` | 任务拆解 + 验收命令 |
| `architecture.md` | 技术规范锁定 |
| `implement.md` | 执行手册 |
| `documentation.md` | 进度日志 |

## 使用流程

1. `/plan5` 初始化
2. 编辑 `prompt.md` 定义需求
3. 编辑 `plans.md` 拆解任务
4. 说"开始执行"
5. 完成后更新 `documentation.md`

## 相关命令

- `/plan5 [项目路径]` - 启动五文件工作流
