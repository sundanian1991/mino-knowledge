## Tavily 搜索技能安装（2026-03-20）
- 技能：openclaw-tavily-search（Tavily 搜索）
- 安装方式：离线等价安装（ClawHub 限流）
- 文件写入：`~/.joyclaw/builtinSkills/openclaw-tavily-search/`
- 功能：通过 Tavily API 进行网络搜索（Brave 搜索的替代方案）
- 配置：需要 Tavily API Key（环境变量 `TAVILY_API_KEY` 或 `~/.openclaw/.env`）
- 使用示例：
  ```bash
  python3 ~/.joyclaw/builtinSkills/openclaw-tavily-search/scripts/tavily_search.py --query "OpenClaw 技能" --max-results 5
  ```
