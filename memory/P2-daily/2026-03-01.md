## 技术细节

### Viewport 适配实现
```css
.slide {
    width: 100vw;
    height: 100vh;
    height: 100dvh;  /* 动态视口高度 */
    overflow: hidden; /* 防止任何滚动 */
}
```

### 响应式排版
- 字体使用 `clamp(min, vw, max)` 响应式缩放
- 断点：700px / 600px / 500px 短视口适配
- 网格在小屏幕自动变单列

### 导航支持

- 鼠标滚轮滚动导航
- 右侧导航点点击跳转
