## 对话摘要

### 用户需求
将 `workspace/07-投影大纲（5 页）.md` 的内容制作成会议投影演示文稿。

### 执行过程

1. **读取大纲内容** — 定价委员会机制策略组讨论，5 页投影大纲
2. **风格确认** — 用户选择：
   - 用途：内部会议投影
   - 风格选择：看预览后选择
3. **生成 3 个预览**：
   - Style A: Professional Corporate（深蓝专业风）
   - Style B: Modern Tech（深色科技风）
   - Style C: Clean Minimal（清爽极简风）
4. **用户选择**：Style C 和 Style A，要求生成完整 5 页
5. **生成两份完整演示文稿**：
   - `presentation-corporate.html` — 6 页（深蓝专业风）
   - `presentation-minimal.html` — 6 页（清爽极简风）
6. **大纲内容更新** — 用户修改了原始大纲（问题描述、组织架构、表格格式等）
7. **重新生成** — 根据更新后内容，用 Style C 重新生成最终版

### 核心内容（投影大纲）

| 页码 | 主题 |
|------|------|
| 1 | 封面标题 |
| 2 | 背景与目标（当前问题 vs 建设目标） |
| 3 | 组织架构（5 人委员会 + C2 终审） |
| 4 | 核心流程（5 阶段 + 三条红线表格） |
| 5 | 策略组的角色（四大职责 + 时间投入） |
| 6 | 下一步（3 月试运行计划 + 会后行动） |

### 技术实现

- **viewport 适配** — 每页精确 100vh/100dvh，无滚动
- **响应式排版** — clamp() 字体/间距缩放
- **短视口适配** — 700px/600px/500px 断点
- **导航方式** — 方向键、滚轮、触摸、右侧导航点
- **动画** — CSS reveal 动画，支持 prefers-reduced-motion
